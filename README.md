## build

### windows
````
mkdir build
cd build
cmake .. -G"Visual Studio 15 2017 Win64"
````

### android

使用android studio 打开./android/目录即可

若想增加android测试例子，见./buildsys/android目录下的generate_sample_gradle.bat
